# CM SIG组

CM，集群管理系统，为opengauss数据库提供了主备的状态监控、网络通信故障监控、文件系统故障监控、故障自动主备切换等能力。同时提供了丰富的集群管理能力，如集群、节点、实例级的启停，集群状态查询、主备切换、日志管理等。相信Cluster Manage SIG的成立，可以使集群的可靠性更上一层楼。

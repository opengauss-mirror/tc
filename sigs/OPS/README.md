# OPS
OPS SIG 聚焦openGauss维护能力建设，提升产品运维能力，总结和传递运维经验, 包括不限于:
（1）健康监控：关键指标监控，如语句响应时延时、TPS、QPS、会话、Buffer命中率等
（2）性能调优：慢SQL日志、PlanHint
（3）故障处理：CORE堆栈，黑匣子，热补丁
（4）AI智能运维：AI技术提升运维能力

# Graph
Graph SIG 致力于构建统一存储和查询的知识图谱数据管理功能，包括基于openGauss平台实现RDF图和属性图两大知识图谱数据模型的存储方案共享和查询互操作，Cypher和SPARQL两大知识图谱查询语言的语义对齐和语法等价转化。
